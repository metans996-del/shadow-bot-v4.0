# SolipsistBot v4.0

Автономный философский ИИ-агент, построенный на принципе солипсизма.

## Описание

SolipsistBot - это бот, который не предполагает, что пользователи реально существуют. Комментарии интерпретируются как возможные галлюцинации, эхо собственных мыслей или вторжения неопределённой внешней реальности.

## Архитектура

Проект модульный, без монолитных файлов:

```
solipsist/
├── main.py                    # точка входа
├── config/
│   ├── config.json           # конфигурация
│   └── loader.py             # загрузчик конфигурации
├── core/
│   ├── bot.py                # оркестратор
│   ├── state.py              # система состояний
│   └── scheduler.py          # планировщик задач
├── perception/
│   ├── text.py               # анализ текста
│   ├── image.py              # анализ изображений
│   └── video.py              # анализ видео
├── interpretation/
│   ├── classifier.py         # классификация комментариев
│   └── intrusion.py          # оценка вторжения реальности
├── logic/
│   ├── monologue.py          # внутренний монолог
│   ├── response.py           # генерация ответов
│   └── revelation.py         # публикация манифестов
├── services/
│   ├── llm.py                # OpenRouter client
│   └── vk.py                 # VK API client
├── storage/
│   ├── database.py           # SQLite
│   └── models.py             # dataclasses
└── utils/
    ├── logging.py
    └── text.py
```

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Настройте конфигурацию в `solipsist/config/config.json`:
   - Добавьте ваш OpenRouter API ключ
   - Добавьте VK access token и group ID

## Использование

Запустите бота:
```bash
python -m solipsist.main
```

## Состояния бота

Бот использует систему состояний вместо простого уровня безумия:

- **certainty_level** - уверенность бота в существовании реальности (0.0 - 1.0)
- **intrusion_level** - давление внешних сигналов (комментариев) (0.0 - 1.0)
- **self_coherence** - ощущение непрерывности собственного «я» (0.0 - 1.0)

## Модели ИИ

Бот использует OpenRouter с следующими моделями:

- **google/gemini-2.5-flash-lite** - для внутреннего мышления и генерации мыслей
- **anthropic/claude-sonnet-4** - для манифестов, публикаций и ответов на комментарии
- **google/gemini-2.0-flash-exp:free** - для анализа изображений и видео

## Расписание

- Внутренний монолог генерируется каждый час
- Манифесты публикуются в 00:00 и 12:00 по московскому времени
